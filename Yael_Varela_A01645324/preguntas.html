Preguntas HTML y CSS

¿Qué es HTML?

HTML es un lenguaje de marcado que da estructura a la información contenida dentro de una página web

Explica las etiquetas de head y body.

La etiqueta body es aquella que se usa para estructurar el contenido principal de la página web, es decir, la información que se desea hacer visible en la página web, como texto, imágenes o enlaces. Mientras que la etiqueta head sirve para estructurar la información de la página, pero que no se verá visible para el usuario, como podrían ser el título de la pestaña, enlaces de estilo CSS, etc.

Explica las etiquetas section, article, aside, header y footer.

Section: Representa una sección de contenido relacionado, es decir, que corresponde a un mismo tipo de tema

Article: Representa una seccion especifica de contenido independiente, es decir, un articulo

Aside: Representa una sección de información que tiene relación al contenido principal de la página web pero que se puede separar, es decir, son fragmentos de información que complementan la información principal.

Header: Es la primera sección de información que ven los usuarios en la parte superior de la página, es decir, se utiliza esta etiqueta para mostrar el encabezado.

Footer: Es la etiqueta que simboliza el pie de página, es decir, la sección de contenido que se ubica en la parte inferior de la página web.

CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) se trata de un lenguaje de diseño gráfico el cual permite personalizar una página web, permitiendo crear hojas de estilo con el cual controlar la apariencia de una pagina, como el color, la alineación del contenido, el tipo de letra y su tamaño, etc.

¿Cuál es su proposito?

Su proposito es el de añadir estilos y personalización a una página web. Separando la información de la presentación visual con la cual fácilitar el mantenimiento y la legibilidad del desarrollo de la página.

¿Como aplicas CSS a HTML (3 opciones)?

Mediante el uso de la etiqueta style de forma separada en head.

Mediante la etiqueta style pero en la linea que se desea personalizar. Utilizando style dentro del inicio de la etiqueta de parrafo (p) o titulo.

Mediante un archivo separado .css y se enlaza al head del documento html mediante la etiqueta link

¿Qué es un selector y que es una regla CSS?

Un selector son aquellos que delimitan o seleccionan el o los elementos del contenido de la página a los cuales aplicar estilos.

Una regla CSS son las declaraciones que consisten de un selector con el cual seleccionan el contenido a editar y un bloque de propiedades y valores con las cuales se cambiaran el contenido, por ejemplo el color o el tamaño de la letra.

Explica los selectores: Tag selector, Class Selector, Tag and Class Selector, Id Selector

Tag Selector: Selecciona los elementos por su etiqueta, por ejemplo p{} selecciona los elementos dentro de

(todos los parrafo)

Class Selector: Selecciona todos los elementos que pertenezcan a una clase, utilizando un punto (.) antes del nombre de la clase, ejemplo, .nombreDeClase{}

Tag and Class Selector: Combina el Tag Selector y el Class Selector, donde selecciona elementos especificos, por ejemplo, si usaramos el ejemplo de

(todos los parrafos) al usar p.clase{} solo se seleccionarian los parrafos p con la clase seleccionada

Id Selector: Se selecciona el elemento por su atributo que funge como su identificador, esto se utiliza usando el simbolo "#". #id{}

¿Qué es una clase y id? ¿Cómo se relacionan con CSS?

Una clase es un nombre seleccionado de manera arbitraria que guardara un estilo personalizado y que se aplicara a un conjunto de contenido deseado, es decir, a multiples elementos.

El Id es un atributo que se asigna a un solo elemento y es seleccionado mediante el simbolo #, esto con el fin de crear un identificador único para dicho elemento.

Las clases y los id se relacionan a CSS en sentido de que permiten aplicarle estilos personalizados a los elementos HTML que se seleccionen mediante el uso de clases y id.

¿Cómo puedes cambiar el color de la fuente con CSS?

Esto se puede hacer usando la propiedad "Color: -el nombre del color que se desee-" la cual permite seleccionar colores mediante su nombre (como Red, Yellow, Lime, etc.) o mediante el uso de sus valores RGB, hexadecimales y HSL.
