Actividad 8: Cats API

¿Qué API se esta utilizando?

Utilizando la API "Cataas" es posible el poder obtener una coleccion de imagenes y gifs sobre este animal al cual seleccione porqué es lindo.

Go to Cataas

Tutorial

  1. El primer paso es obtener la URL de la API al dar clic en el botón "Go to Github" en esta página.
  2. Después se deberá crear un archivo JavaScript donde se manejará la lógica de la API.
  3. Dentro del archivo JavaScript elige un endpoint para obtener una imagen o un dato.
  4. Programa una función para obtener tu endpoint deseado, utilizando "fetch" el cual por defecto utiliza el método GET.
  5. Configura la función para procesar la respuesta utilizando blob ya que el formato de esta api no devuelve un JSON, y, extrae los datos necesarios (como imágenes o gifs).
  6. Muestra los datos obtenidos en tu página HTML, insertándolos en los elementos correspondientes.

endpoints

  1. El endpoint base "https://cataas.com/cat" te regresa una imagen random de un gato
  2. El endpoint "https://cataas.com/cat/gif" te regresa un gif random de un gato
